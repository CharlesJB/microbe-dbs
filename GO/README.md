
This is for building input files for gene ontology profiling
with Ray Ontologies.

	GO-Main.sh [output directory]

This will generate these files :
* EMBL_CDS_Sequences/
* 000-Annotations.txt
* 000-Ontologies.txt
* 000-Sequences.fasta
* Annotations.txt
* Associations.txt
* cds.fasta
* gene_association.goa_uniprot
* gene_ontology_ext.obo
* idmapping.dat
* OntologyTerms.txt
* Sequences-With-Annotations.fasta
