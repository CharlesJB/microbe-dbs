# microbe-dbs

Utilities to fetch biological databases relevant for microbial genomics analysis.

```
$ bash microbe-dbs.sh -h
 microbe-dbs.sh databases output
    
    databases :
      -> refseq_bacteria
      -> refseq_plasmid
      -> refseq_viral

```
